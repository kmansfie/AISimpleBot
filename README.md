# 🤖 AI Projects

A collection of AI and robot projects organized by platform and purpose.

## 📁 Project Structure

```
ai/
├── README.md              # This file
├── .gitignore             # Python ignore rules
├── simplebot/             # Simple robot skeleton
│   ├── README.md
│   ├── .gitignore
│   ├── requirements.txt
│   ├── LICENSE
│   ├── pc_code/           # PC control scripts
│   ├── rpi_code/          # Raspberry Pi code
│   ├── prompts/           # LLM prompts
│   └── logs/              # Exploration logs
├── robot-exploration/     # Full robot exploration project
├── vision/                # Computer vision projects
├── nlp/                   # Natural language projects
└── utils/                 # Shared utilities
```

## 🚀 Getting Started

Each sub-project is a separate git repository. Check out the individual README files for details.

### Simple Bot

The `simplebot/` directory is a skeleton project you can use as a template:

```bash
cd ai/simplebot
# Add your files
git add .
git commit -m "Initial commit"
git push
```
